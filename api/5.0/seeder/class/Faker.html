readonlyfake

fake: (str: string) => string

Type declaration

    • (str: string): string

    • Generator method for combining faker methods based on string input

      Example:

      console.log(faker.fake('{{name.lastName}}, {{name.firstName}} {{name.suffix}}'));
      // outputs: "Marks, Dean Sr."

      This will interpolate the format string with the value of methods [name.lastName]faker.name.lastName, [name.firstName]faker.name.firstName, and [name.suffix]faker.name.suffix

      faker.fake
